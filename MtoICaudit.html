Manager to IC Audit Tool

This tool is used to easily create a report that can be sent to Peeps Partners, notifying them of Managers without subordinates.

Select the source master file that (CSV or XLSX) that contains the primary information.
Select the source exception file (CSV or XLSX) that contains the exception information.
Select the mTeam Member to build the report from
Select the mTeam Directs that you wish to process this report on.
Scroll down to see the results.
Take a moment to review the results before downloading the output file.
Download your results file.
